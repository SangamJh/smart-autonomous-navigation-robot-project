# Smart Autonomous Navigation Robot

## Overview
This project is an Arduino-based robot that detects obstacles using an ultrasonic sensor and automatically chooses the best path to avoid collisions.

## Features
- Obstacle detection using HC-SR04
- Intelligent left/right path decision
- Motor control using L298N driver
- Modular and clean code structure

## Components
- Arduino UNO
- Ultrasonic Sensor (HC-SR04)
- L298N Motor Driver
- DC Motors
- Servo Motor
- Battery

## Project Structure
- code/ → Arduino code
- images/ → project photos
- circuit/ → wiring diagram
